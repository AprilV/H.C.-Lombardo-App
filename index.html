Flask API — EC2

  • Port 5000 · 26 REST endpoints
  • api_server.py, api_routes_hcl.py, api_routes_ml.py, api_routes_live_scores.py
  • Systemd service · auto-pulls from GitHub
  • CORS enabled · background updater service

PostgreSQL — EC2

  • Port 5432 · Schema: hcl
  • 7,269 games · 14,398 team-game records
  • 64 stats/game including EPA
  • Tables: games, team_game_stats, ml_predictions, betting_lines

ESPN API

  • External · HTTPS (443)
  • Live scores and standings
  • Fetched by Flask, cached in DB
  • Background updater runs on schedule

ML Engine

  • XGBoost winner classifier (46 features)
  • XGBoost spread regressor
  • Elo rating system (elo_tracker.py)
  • Ensemble predictions vs Vegas

NFLverse

  • nfl_data_py Python package
  • 7,269 games play-by-play (1999–2025)
  • EPA per play · 64 advanced stats/game
  • Loaded offline → PostgreSQL hcl schema
  • Basis for all ML model training

Infrastructure

  • Frontend: AWS Amplify (auto-deploy)
  • Backend: AWS EC2 (systemd service)
  • Database: PostgreSQL 18 on EC2
  • CI/CD: GitHub → Amplify + EC2
  • Data: NFLverse (play-by-play) + ESPN API

Disclaimer

This application is for educational and research purposes only. NFL predictions, betting analysis, and statistical models are academic exercises and should not be used for financial decisions.


Data sourced from NFLverse (open source) and ESPN public API.

Application Features
📊
Historical Analytics
7,263 games from 1999–2025. EPA metrics, weather impact, rest advantage, referee tendencies, and ATS betting performance.
🤖
ML Predictions
XGBoost winner classifier + spread regressor using 46 rolling features. Elo ratings system. Ensemble predictions vs. Vegas lines.
📡
Live Scores
Real-time game data via ESPN API, overlaid with AI predictions, Elo ratings, and Vegas spread comparisons.
🏈
Team Tools
Head-to-head team comparison, individual team detail pages, matchup analyzer, and custom stat builder.
📈
Model Performance
Season-long tracking of AI predictions vs. actual results vs. Vegas. Accuracy metrics, MAE, and ATS record.
⚙️
Admin Panel
Data refresh controls, background updater management, and database health monitoring.
