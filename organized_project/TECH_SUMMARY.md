# H.C. Lombardo App - Technology Summary

## 🔧 **Technologies Used in This Project**

### **Python Backend** 🐍
- **FastAPI** framework for REST APIs
- **SQLite** database integration  
- **Jinja2** template rendering
- **uvicorn** ASGI server
- Machine learning libraries for text classification

**Main Files:**
- `hc_lombardo_dashboard.py` (Primary application)
- `fastapi_template_inheritance.py`
- `nfl_betting_api.py`
- `text_classification.py`

---

### **HTML Templates** 📄
- **Jinja2** template inheritance system
- **Responsive HTML5** structure
- **Component-based** template architecture
- **Dynamic content** rendering with template variables

**Main Files:**
- `base.html` (Master template with sidebar)
- `working-sidebar.html` (Standalone test)
- `teams.html` (NFL teams page)
- `predictions.html` (NFL predictions page)

---

### **CSS Styling** 🎨
- **Modern CSS3** with animations and transitions
- **Flexbox & Grid** layouts
- **Responsive design** with media queries
- **Glass morphism** effects and gradients
- **Professional color scheme** (Gold #FFD700 and blues)

**Main Files:**
- `base.css` (Core styles)
- `navigation.css` (Sidebar styling)
- `themes.css` (Color themes)

---

### **JavaScript Logic** ⚡
- **Vanilla JavaScript ES6**
- **DOM manipulation** for sidebar functionality  
- **API communication** with fetch()
- **Event handling** and animations
- **Local storage** and state management

**Main Files:**
- `main.js` (Core functionality)
- `api-client.js` (API communication)

---

## 🏗️ **Architecture Pattern**

**Self-Contained FastAPI Application**
- All HTML/CSS/JS embedded in Python app
- No external template dependencies
- Single-file deployment capability
- Professional sidebar navigation system

**Database Integration**
- SQLite for data persistence
- Mock data fallback system
- NFL statistics and betting data
- Real-time data capability
